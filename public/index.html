Brainbase: The future of chatbots

Brainbase is a web-based AI chatbot.



Brainbase uses models to run the text completions.
We have a (Broken, non-production) API you can use, although there is no documentation.
Simply use /chatbotresponse?question=Message with these self-explanitory args:
&swear=true, &slang=true, &long=true, &character=human (or not human), &personality=x
Also, make sure to add your 'chat history' by adding the HTTP Body as {history: x} (with x being a stringified array of strings).
